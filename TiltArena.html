Tilt Arena

Avoid the bullets and survive as long as possible!
GitHub Repository


Built with the intention to learn JavaScript and Canvas, added a Firebase NoSQL database to keep track of player data
Built With:
-HTML, JavaScript (Canvas, Render & RequestAnimationFrame, KeyListeners), Firebase, NoSQL
Concepts:
-Firebase stores player data by name such as deaths, high score, etc.
-Also stores global data such as plays, total deaths, total enemies killed, etc
-Having a DeltaTime that keeps track of the time between frames
---DeltaTime is sent to all objects that move or have animation
---By using DeltaTime in calculations with animations, gameplay experience is the same even if there is a long time between frames
-Extends for Bomb and Enemy objects
-Multiple enemy AI's and Bomb types to use.
