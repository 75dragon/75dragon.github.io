Vast

A Java Mining game!
GitHub Repository


Built in APCS at Lynbrook High School, updated and cleaned throughout the years.
Built With:
-Java (Swing, Timer, Event, ImageIO, Color)
Concepts:
-Cellular automata in cave generation.
-Dijkstra's algorithm for optimal AI pathing.
-Data structures to hold map, items, enemies, etc.
-Extends and Implements for various similar objects.
-Multiple enemy AI's, weapons, pickups to find.
Complied JAR located in the GitHub repository.
